VS Code 强大的插件生态让它成为最受欢迎的代码编辑器。以下是我推荐的必装插件。

## 基础增强

### Prettier - Code Formatter
自动格式化代码，保持代码风格一致。

### ESLint
JavaScript/TypeScript 代码检查，及时发现潜在问题。

### GitLens
强化 Git 功能，查看代码历史和责任归属。

## 效率提升

### Auto Rename Tag
同步修改 HTML/JSX 标签名。

### Bracket Pair Colorizer
用不同颜色标记配对的括号。

### Path Intellisense
自动补全文件路径。

## 主题美化

### One Dark Pro
经典的深色主题。

### Material Icon Theme
漂亮的文件图标。

## AI 辅助

### GitHub Copilot
AI 代码助手，智能补全代码。

### Codeium
免费的 AI 代码补全工具。

选择适合自己的插件，让开发更加高效愉快！
