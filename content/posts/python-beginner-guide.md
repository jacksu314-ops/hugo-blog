Python 是一门简洁优雅的编程语言，非常适合初学者入门。

## 为什么选择 Python？

- **语法简洁**：接近自然语言
- **应用广泛**：Web、数据科学、AI、自动化
- **社区活跃**：丰富的库和教程

## 第一个程序

```python
print("Hello, World!")
```

## 变量和数据类型

```python
# 字符串
name = "Alice"

# 数字
age = 25
height = 1.65

# 布尔值
is_student = True

# 列表
hobbies = ["reading", "coding", "music"]
```

## 条件判断

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("需要努力")
```

## 循环

```python
# for 循环
for i in range(5):
    print(i)

# while 循环
count = 0
while count < 3:
    print(count)
    count += 1
```

## 函数

```python
def greet(name):
    return f"Hello, {name}!"

message = greet("World")
print(message)
```
