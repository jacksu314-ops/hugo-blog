滑动轴承计算里，一个看似简单却很容易被做错的问题是：**给定转速与外载荷后，轴颈中心最终停在哪里？** 这个位置不仅决定最小油膜厚度，也是后续识别油膜刚度、阻尼和判断稳定性的起点。

本文记录我使用 Fluent 与 UDF 构造静平衡搜索的一套思路。它不是某个模型的固定参数配方，而是一套可以迁移到不同介质、结构和网格上的求解框架。

## 1. 先把问题写成残差方程

设轴颈中心位置为 $\mathbf q=[x,y]^T$，CFD 在该位置给出的流体合力为 $\mathbf F_f(\mathbf q)$，重力及其他外载荷之和为 $\mathbf F_e$。静平衡条件是：

$$
\mathbf R(\mathbf q)=\mathbf F_f(\mathbf q)+\mathbf F_e=\mathbf 0
$$

这里真正要收敛的是**力残差与位置增量**，而不只是 Fluent 的连续性残差。流场残差下降只说明当前网格位置上的离散方程趋于稳定，并不能证明轴颈已经处于平衡点。

油膜力需要对轴颈壁面压力和黏性应力积分：

$$
\mathbf F_f=\int_A(-p\mathbf n+\boldsymbol\tau\cdot\mathbf n)\,\mathrm dA
$$

第一轮调试时应分别输出压力项、黏性项、合力方向及坐标变换结果。符号、法向和旋转坐标系一旦混淆，搜索过程可能“稳定地走向错误方向”。

## 2. 用 Newton 第二定律构造搜索器

直接求解 $\mathbf R(\mathbf q)=0$ 可以采用有限差分 Jacobian 与 Newton 迭代；在 Fluent 动网格流程中，也可以引入虚拟质量和数值阻尼：

$$
\mathbf M_v\ddot{\mathbf q}+\mathbf C_v\dot{\mathbf q}=\mathbf R(\mathbf q)
$$

$\mathbf M_v$ 与 $\mathbf C_v$ 是搜索参数，不应被误解为真实转子的质量和阻尼。阻尼过小会在平衡点附近振荡，反馈增益过大则可能跨过最小间隙并产生负体积。工程上更稳妥的做法是限制单步位移：

$$
\lVert\Delta\mathbf q\rVert\leq \eta c
$$

其中 $c$ 为径向间隙，$\eta$ 应从较小值开始测试。

## 3. `CG_MOTION` 与 `GRID_MOTION` 怎么选

- `DEFINE_CG_MOTION` 适合把轴颈边界视作刚体，UDF 返回线速度和角速度。
- `DEFINE_GRID_MOTION` 适合需要逐节点更新、边界本身发生非刚性变形，或必须精细控制节点位置的情况。

静平衡搜索通常是轴颈整体平移，因此刚体运动往往更简洁。若使用 `GRID_MOTION`，必须只更新目标动态区域的节点，并用节点更新标记避免同一节点在一个时间步被重复移动。

```c
DEFINE_GRID_MOTION(journal_motion, domain, dt, time, dtime)
{
    /* 读取上一步经过并行归并的壁面合力 */
    /* 根据受力残差计算受限位移增量 */
    /* 遍历动态边界节点并更新坐标 */
    /* 记录位置、合力、最小间隙和迭代号 */
}
```

不要在运动宏中混入过多流场计算。更清晰的分工是：在合适的 execute-at-end 宏中积分和归并力，在运动宏中只消费已经准备好的状态。

## 4. 稳态与瞬态如何交替

动网格需要瞬态求解器，但目标是静平衡。可采用“冻结位置求流场，更新位置再松弛”的外层迭代：

{{< mermaid >}}
flowchart LR
  A[初始化同心或经验位置] --> B[固定位置收敛流场]
  B --> C[积分油膜力]
  C --> D[计算受限位移]
  D --> E[更新网格并检查质量]
  E --> B
  C --> F{力与位移均收敛?}
  F -->|是| G[保存平衡位置]
{{< /mermaid >}}

每次移动后必须给流场足够的内迭代次数。若位置更新比流场松弛更快，搜索器读取的就是滞后的油膜力，极易形成数值振荡。

## 5. 我采用的收敛判断

至少同时检查四项：

1. 相对力残差 $\lVert\mathbf R\rVert/\lVert\mathbf F_e\rVert$ 小于阈值。
2. 连续若干外层迭代的位置增量 $\lVert\Delta\mathbf q\rVert/c$ 足够小。
3. 当前固定位置下，流量、壁面力和连续性残差均已稳定。
4. 最小间隙为正，网格正交质量和体积没有持续恶化。

单点满足条件不算收敛，应该要求这些条件连续成立若干次。

## 6. 最常见的踩坑

- **力方向错误**：先用静压或对称工况验证积分方向。
- **把流场残差当总收敛**：额外记录力残差和轴心轨迹。
- **位移步长过大**：对增量限幅，并在接近平衡时降低增益。
- **并行结果不一致**：壁面力必须在计算节点间正确归并，再传给主节点。
- **网格只看初始质量**：监控整个搜索过程中的最小体积与最小间隙。
- **时间尺度混乱**：区分真实瞬态响应和用于寻找静平衡的伪时间推进。

最终可信的结果不只是一个 $(x,y)$ 坐标，而应包含网格无关性、步长敏感性、载荷平衡误差和轴心搜索轨迹。只有这些证据同时成立，静平衡位置才有资格进入后续动力学分析。
