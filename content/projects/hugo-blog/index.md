这是一个使用 Hugo 和 Blowfish 构建、由我独立设计与维护的个人数字花园。它同时承载工程研究、技术文章、项目案例和语言学习，并通过 Supabase 提供身份验证、讨论与私人学习记录同步。

## 项目背景

最初的需求很简单：把零散的技术笔记整理成可以搜索和归档的网站。随着内容增加，首页排版、项目边界、移动端导航、文章日期统计和持续更新记录都成为需要解决的真实问题。

## 我的工作

- 重新设计首页内容面板、文章卡片和项目卡片
- 修正归档链接、导航入口与发文热力图日期匹配
- 建立每日学习、周复盘和版本修改日志
- 接入 Supabase Auth、PostgreSQL 与 RLS，建立公开阅读、验证用户和管理员三层权限
- 建立站内讨论区，并支持评论回复、本人编辑与管理员审核
- 对桌面端与移动端持续进行响应式检查

## 核心能力

- 内容：文章、标签、分类、归档与全文搜索
- 学习：每日任务、完成统计、连续打卡和周复盘
- 互动：验证用户评论、回复与版本讨论
- 阅读：目录、阅读进度、代码复制和随机探索
- 展示：项目案例、技术栈与迭代状态

## 关键取舍

网站继续采用静态生成，保持部署简单和页面加载稳定。公开内容由 GitHub Pages 托管，身份、评论和私人学习记录由 Supabase 管理，并通过数据库 RLS 保证权限不只依赖前端按钮隐藏。

## 下一步

- 增加月度学习趋势、模考成绩与每日训练结果
- 持续补充旋转机械、CFD 和工程计算文章
- 为真实工程项目增加图表、指标和复盘
- 继续压缩资源体积并改善低性能设备体验
