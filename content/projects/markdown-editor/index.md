该项目目前处于概念规划阶段，目标是提供一个打开即写、无需注册、不会轻易丢失草稿的 Markdown 编辑空间。

## 使用场景

适合快速撰写博客初稿、README 和带代码片段的技术记录。桌面端可以使用编辑与预览双栏，移动端则切换为单栏标签页。

## 计划能力

- Markdown 实时预览
- 代码块语法高亮
- 本地自动保存与草稿恢复
- 双栏、编辑、预览三种模式
- 导出 Markdown 与 HTML

## 交互重点

编辑器需要保持稳定尺寸，切换预览模式时不改变滚动位置。工具栏使用常见格式图标，并为不熟悉的操作提供悬停提示。

## 成功标准

长文编辑过程中输入不卡顿，刷新页面后可以恢复草稿，移动端工具栏和正文不会互相遮挡。
